# 🖥️ System Monitoring Tool

A simple Python-based system monitoring script that displays CPU, memory, and disk information.

## 🚀 Features
- Displays CPU information
- Shows total physical memory
- Lists files (disk info)

## 🛠️ Technologies Used
- Python
- OS module

## ▶️ How to Run
1. Open terminal
2. Navigate to project folder
3. Run:
   python system_monitor.py

## 📚 Learning Outcome
- Understanding system-level commands
- Interaction between Python and OS
- Basics of system monitoring
